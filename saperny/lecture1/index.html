Как насчёт новых технологий?

Как правило относят:

  • JavaScript(AJAX & Comet)
  • CSS3
  • HTML5

JavaScript, AJAX, Comet

JavaScript — C-подобный язык, который может исполняться прямо в браузере, до некоторого времени был единственной возможностью создания динамически web-страниц.

AJAX(assynchronus javascript and xml) — технология, с помощью которой JavaScript может общаться с удалённым сервером, отправлять HTTP запросы.

Comet — технология, которая позволяет серверу посылать данные клиенту, а не только отвечать. Понятие частично включает в себя AJAX

CSS3. Конец кошмара.

Улучшения по сравнению с CSS2:

  • Работа над ошибками. Сложные техники заменены на стандартные решения.
  • Появились функции для работы с анимацией и передвижением объектов.
  • CSS3 быстрее и надёжнее JavaScript (!)
